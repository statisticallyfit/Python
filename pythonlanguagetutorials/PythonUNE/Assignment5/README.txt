How to run the program:
* step 1: unzip the a5.tgz file by typing this in the command line in the same directory
as the a5.tgz:
  tar -zxf a5.tgz

* step 2: make sure that all the files are in your location: clean.py

* step 3: run the file by typing:
python3 clean.py

* step 4: input the required information as the program asks. In this case, click on
the buttons that appear in the window and watch the average rating appear and adapt to new input.
