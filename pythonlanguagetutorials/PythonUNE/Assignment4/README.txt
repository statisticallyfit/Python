How to run the program:
* step 1: unzip the a4.tgz file by typing this in the command line in the same directory
as the a4.tgz:
  tar -zxf a4.tgz

* step 2: make sure that all the files are in your location: routes.py, routes.txt, README.txt.

* step 3: run the file by typing:
python3 routes.py

* step 4: input the required information as the program asks.
