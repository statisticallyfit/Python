How to run the program:
* step 1: unzip the a3.tgz file by typing this in the command line in the same directory
as the a3.tgz:
  tar -zxf a3.tgz

* step 2: go inside the a3 directory with this command:
cd a3

* step 3: run the file by typing:
python3 busstop.py

* step 4: input the required information as the program asks.
